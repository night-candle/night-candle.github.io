    public int removeDuplicates(int[] nums) {
    int i = nums.length > 0 ? 1 : 0;
    for (int n : nums)
        if (n > nums[i-1])
            nums[i++] = n;
    return i;
}
//过于晦涩
    //双指针
public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;

    int slow = 0, fast = 0;
    while (fast < nums.length) {
        if (nums[fast] != nums[slow]) {
            slow++;
            // 维护 nums[0..slow] 无重复
            nums[slow] = nums[fast];
        }
        fast++;
    }
    // 数组长度为索引 + 1
    return slow + 1;
}

    public int[] plusOne(int[] digits) {
    for (int i = digits.length - 1; i >= 0; i--) {
        if (digits[i] < 9) {
            digits[i]++;
            return digits;
        }
        digits[i] = 0;
    }
	// 最高位需要进位
    digits = new int[digits.length + 1];
    digits[0] = 1;
    return digits;
}

    排序 + 双指针 | 哈希

    // 一次哈希
public int[] twoSum(int[] nums, int target) {
    HashMap<Integer, Integer> hashMap = new HashMap<>();

    for (int i = 0; i < nums.length; i++) {
        if (hashMap.containsKey(target - nums[i]))
            return new int[]{hashMap.get(target - nums[i]), i}; //表中存过
        else
            hashMap.put(nums[i], i);
    }
    return null;
}

  • LeetCode15. 三数之和

    双指针

    public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> ans = new ArrayList<>();
//        if (nums == null || nums.length <= 2) return ans;

    Arrays.sort(nums); // 排序
    for (int i = 0; i < nums.length - 2; i++) { // O(n^2)
        if (nums[i] > 0) break; // 第一个数大于 0，后面的数都比它大，肯定不成立了
        if (i > 0 && nums[i] == nums[i - 1]) continue; // 去掉重复情况

        int target = -nums[i]; //转化为两数之和问题

        int left = i + 1, right = nums.length - 1; //双指针
        while (left < right) {
            int sum = nums[left] + nums[right];
            if (sum == target) {
                ans.add(new ArrayList<>(Arrays.asList(nums[i], nums[left], nums[right])));
                left++;
                right--;
                //去掉重复情况
                while (left < right && nums[left] == nums[left - 1]) left++;
                while (left < right && nums[right] == nums[right + 1]) right--;
            } else if (sum < target) {
                left++;
            } else {  // nums[left] + nums[right] > target
                right--;
            }
        }
    }
    return ans;
}

    固定最左的指针 k，双指针 i，j 分设在数组索引 [k,len(nums)] 两端，通过双指针交替向中间移动，记录对于每个固定指针 k 的所有满足 nums[k] + nums[i] + nums[j] == 0 的 i，j 组合

    1. 当 nums[k] > 0 时直接break
    2. 当 k > 0且nums[k] == nums[k - 1]时即跳过此元素nums[k]（去重）
    3. i < j 时循环计算 s = nums[k] + nums[i] + nums[j]，并移动双指针

    public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;

    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;

        if (slow == fast) return true; //有环
    }
    return false;
}

  • 142. 环形链表 II

    public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;

    while (fast != null && fast.next != null){ //注意条件！
        slow = slow.next;
        fast = fast.next.next;
        if (fast == slow) break; //相遇
    }

    if (fast == null || fast.next == null) return null;

    //重置指针
    slow = head;
    while (fast != slow){
        slow = slow.next;
        fast = fast.next;
    }
    return slow;
}

AVL 树

  • 一颗自平衡的二叉搜索树，AVL 树引入了监督机制保证树的平衡度在可以接受的范围内，树的某一部分的不平衡度超过一个阈值后触发相应的平衡操作

    平衡因子（Balance Factor）：某个结点的左子树的高度减去右子树的高度的差值

  • AVL 树具有以下性质

    • 根的左右子树的高度之差的绝对值不能超过1
    • 根的左右子树都是平衡二叉树

  • 插入一个节点可能会破坏 AVL 树的平衡（只有从插入点到根节点路径上的节点平衡可能被破坏），可以通过旋转操作来进行修正；不平衡的出现可能有4种情况

    • 对K的右孩子的右子树插入导致的：左旋
    • 对K的左孩子的左子树插入导致的：右旋
    • 对K的右孩子的左子树插入导致的：右旋-左旋
    • 对K的左孩子的右子树插入导致的：左旋-右旋
